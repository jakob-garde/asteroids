# Astescribble

This project is coded with raylib for drawing, window and input handling.

Code features:

- animations
- collisions
- multiple entities frequently created and destroyed
- asteroid spawn phases
- level phases
- level transitions

Game design:

- a five minute game
- unique art style, feels like the game is scribbled on a notebook
- diegetic, mechanics are discovered during play
- simple sound effects
- music tracks match difficulty
- chill start
- ramping intensity
- difficulty spikes
- cute endscreen
